# Mira brand assets

| File | Use |
|---|---|
| `mira-mark.svg` | Aperture mark, full color (teal + amber pupil). Use on light surfaces. |
| `mira-mark-mono.svg` | Aperture mark, monochrome (uses `currentColor`). Use anywhere a single-color mark is needed; set color via CSS. |
| `mira-lockup.svg` | Mark + wordmark side-by-side. Use as the primary brand lockup. |
| `mira-app-icon.svg` | 512×512 launcher icon with rounded square plate. |

**Colors used**
- Teal (mark, wordmark): `#2A4A4D` / `oklch(0.32 0.025 195)`
- Amber pupil (accent): `#C97A4A` / `oklch(0.62 0.11 55)`
- Plate (app icon): `#FAF8F3`

The wordmark is set in **Instrument Serif Italic**. Make sure the font is installed or embedded when rendering the lockup; if it isn't available the SVG will fall back to Georgia.
